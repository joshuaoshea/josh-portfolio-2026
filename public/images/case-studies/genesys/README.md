# Genesys Case Study Images

Place your case study images in this folder with the following names:

## Required Images

### Hero Section
- `hero.png` - Main hero image (1500x930px recommended, aspect ratio 749:465)

### Solution Section
- `solution-1.png` - Full-width solution overview (1500x844px, 16:9 ratio)
- `solution-2.png` - Left side-by-side image (730x548px, 4:3 ratio)
- `solution-3.png` - Right side-by-side image (730x548px, 4:3 ratio)
- `solution-4.png` - Bottom left image (730x548px, 4:3 ratio)
- `solution-5.png` - Bottom right image (730x548px, 4:3 ratio)

### Discovery Section
- `discovery-analysis.png` - Analysis visual/diagram (1000x800px recommended)

### Ideation Section
- `ideation-1.png` - First ideation sketch/wireframe (1400x788px, 16:9 ratio)
- `ideation-2.png` - Second ideation sketch/wireframe (1400x788px, 16:9 ratio)

### Research Section
- `research-findings.png` - Research findings visual (1000x800px recommended)

## Image Guidelines

### Format
- **Preferred**: PNG or WebP
- **Alternative**: JPG for photographs

### Optimization
- Compress all images before adding (use [TinyPNG](https://tinypng.com/))
- Target file size: Under 500KB per image
- Maintain quality while reducing file size

### Naming Convention
- Use lowercase
- Use hyphens for spaces
- Be descriptive but concise

## Quick Start

1. Export your designs from Figma/Sketch/Adobe XD
2. Resize to recommended dimensions
3. Compress using TinyPNG or similar tool
4. Name according to the list above
5. Place in this folder
6. The website will automatically load them!

## Need Different Images?

If you want to add more images or change the structure, update the image paths in:
`src/pages/CaseStudyPage.jsx` in the `caseStudy.images` object.
